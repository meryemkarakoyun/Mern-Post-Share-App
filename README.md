# 📝 Post Share App

A full-stack web application built with the **MERN** stack (MongoDB, Express, React, Node.js) along with **Redux** and **React Router DOM**. It allows users to register, log in, create, update, and delete their own posts in a clean and simple interface.

## 🚀 Features

- 🔐 **Authentication** – Secure user registration and login system using JWT (JSON Web Tokens).
- ➕ **Create Post** – Easily add new posts with a title and message via a modal interface.
- ✏️ **Edit/Update Post** – Modify existing posts seamlessly.
- 🗑️ **Delete Post** – Remove posts when they are no longer needed.
- 🔄 **State Management** – Centralized application state handling using Redux.
- 🧭 **Client-Side Routing** – Smooth navigation and protected routes handled by React Router DOM.
- 🔔 **Notifications** – Toast notifications for successful or failed actions using React Toastify.

## 🛠️ Tech Stack

### Frontend (Client)
- **React** – Component-based UI library
- **Redux** – Global state management
- **React Router DOM** – Routing and navigation
- **Axios** – HTTP requests for API communication
- **React Toastify** – Notifications
- **Vite** – Fast build tool and development server

### Backend (Server)
- **Node.js** – JavaScript runtime environment
- **Express.js** – Web framework for building RESTful APIs
- **MongoDB & Mongoose** – Database and ODM for data persistence
- **JWT (jsonwebtoken)** – Secure authentication management
- **Bcrypt.js** – Password hashing for security
- **Cors & Body-Parser** – Middleware for handling requests and cross-origin security
